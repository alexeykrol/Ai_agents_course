# AI Chat Assistant

Современное веб-приложение для общения с искусственным интеллектом с продвинутым пользовательским интерфейсом, персональным анализом и системой авторизации.

## 🚀 Основные возможности

### 💬 Интеллектуальный чат
- **Мгновенные ответы**: Быстрое взаимодействие с ИИ-ассистентом
- **Предустановленные вопросы**: 8 популярных тем для быстрого старта
- **Лимит символов**: До 5000 символов в сообщении с визуальным счетчиком
- **История сообщений**: Сохранение и отображение всей переписки
- **Копирование ответов**: Одним кликом скопировать ответ ИИ

### 🎯 Персональный режим анализа
- **Умный анализ**: Система изучает ваши интересы на основе 10 вопросов
- **Прогресс-бар**: Визуальное отображение прогресса анализа с градиентом
- **Персональные рекомендации**: Индивидуальные вопросы после завершения анализа
- **Категории интересов**: Развитие карьеры, личностный рост, стратегия обучения и др.

### 🔐 Гибкая система авторизации
- **Гостевой режим**: 5 бесплатных вопросов без регистрации
- **Модальная авторизация**: Удобная форма входа/регистрации в модальном окне
- **Email/Password**: Простая регистрация без подтверждения email
- **Восстановление пароля**: Встроенная функция сброса пароля

### 📱 Адаптивный интерфейс
- **Мобильная оптимизация**: Полная поддержка мобильных устройств
- **Коллапсируемый сайдбар**: Умное управление пространством экрана
- **Отзывчивая типографика**: Автоматическая адаптация размеров текста
- **Touch-friendly**: Оптимизированные элементы управления для сенсорных экранов

### ✨ Дополнительные функции
- **Примеры крутых вопросов**: Модальное окно с вдохновляющими идеями
- **Справочная система**: Встроенная помощь с FAQ
- **Индикатор печати**: Анимированный показ активности ИИ
- **Управление чатами**: Создание новых диалогов и навигация по истории

## 🛠️ Технический стек

### Frontend
- **React 18** + **TypeScript** - современный UI фреймворк с типобезопасностью
- **Tailwind CSS** - utility-first CSS фреймворк для быстрой стилизации
- **Vite** - молниеносная сборка и hot reload
- **Lucide React** - красивые SVG иконки
- **date-fns** - работа с датами и временем

### Backend & Database
- **Supabase** - Backend-as-a-Service с PostgreSQL
- **Row Level Security** - встроенная безопасность на уровне базы данных
- **Real-time subscriptions** - мгновенные обновления данных
- **Supabase Auth** - готовая система авторизации

### Deployment
- **Netlify** - автоматический деплой с CI/CD
- **Environment Variables** - безопасное хранение конфигурации

## 🔄 Пользовательские сценарии

### Новый пользователь
1. **Первое посещение**: Видит предустановленные вопросы и информационный блок
2. **Первые 5 вопросов**: Свободное использование без ограничений
3. **6-ой вопрос**: Появляется модальное окно с предложением регистрации
4. **После регистрации**: Полный доступ ко всем функциям

### Персональный анализ
1. **Активация**: Нажатие "Включить мой вопрос"
2. **Сбор данных**: 10 вопросов с визуальным прогрессом
3. **Завершение**: Уведомление о готовности персональных рекомендаций
4. **Использование**: Доступ к индивидуальным вопросам через "Мои вопросы"

### Работа с чатом
1. **Ввод сообщения**: Поддержка до 5000 символов с счетчиком
2. **Отправка**: Enter или кнопка отправки
3. **Получение ответа**: Индикатор печати → ответ ИИ
4. **Копирование**: Кнопка копирования для ответов ИИ

## 🚀 Запуск и разработка

### Локальная разработка
```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build
```

### Переменные окружения
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Структура проекта
```
src/
```
